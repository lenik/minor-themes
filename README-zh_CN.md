# Minor Themes

一组 VS Code 配色主题：柔和粉彩、骄傲旗帜色、复古终端风格，以及植根于各国传统文化经典的色彩表达。

## 主题说明

各主题的色板定义在 `scripts/generate-themes.mjs` 中，与 [UiTheme](https://github.com/lenik) 项目共用同一套调色逻辑。浅色/深色成对的主题保留相同的强调色色相；深色变体在深蓝灰底色上沿用这些强调色。

### Innocent（纯真）

**含义：** *Innocent* — 纯真、清淡、不张扬。

仅提供浅色款。以腮红玫瑰与淡薰衣草为主（色相约 280–340°）：背景近乎白并带一层淡粉，正文为低饱和紫灰。强调色为灰玫瑰与淡紫 — 选区、光标、括号匹配都偏轻盈，不刺眼。语法高亮：关键字玫瑰色，字符串紫灰，数字略深的粉。适合想要安静、偏女性化、长时间阅读不累眼的编辑环境。

### Maiden（少女）

**含义：** *Maiden* — 少女；也指温柔、矜持、略带古典的浪漫气质。

暖调淡粉底色（低饱和红、约 94% 明度），**深玫瑰红**正文，**金黄**高亮（今日标记、字符串、未跟踪文件等）。行号带一点橄榄绿，有老式信纸感。Dark Maiden 在炭灰底上保留玫瑰与金色强调。像淡粉信纸配浓色墨迹 — 表面柔、字可读。

### Girl（女孩）

**含义：** *Girl* — 年轻、鲜明、活泼（偏大胆自信，而非幼稚粉嫩）。

高饱和 **珊瑚红** 侧栏/面板（色相约 4–6°），白底，**热粉**引用块，**天蓝**间隔色。强调色很跳：饱和红光标、青色选区、黄色「今日」标记。Dark Girl 在深色底上保留珊瑚/粉/青的活力。适合喜欢主题有个性的用户。

### Morandi（莫兰迪）

**含义：** 得名于意大利画家 **乔治·莫兰迪（Giorgio Morandi）** — 其静物画以灰调、低饱和、和谐统一著称。

暖 **灰米色** 窗口（色相约 35°、低饱和），**蓝灰** 正文（约 230°）。强调色为灰青与灰粉 — 无荧光感。边框与网格柔和，对比度刻意克制。Dark Morandi 转为冷炭灰底，仍保留莫兰迪式灰青高亮。安静、像画廊，适合长时间编码。

### LGBTQ

**含义：** 取自 **彩虹骄傲旗**（Gilbert Baker 设计）的 stripe 色系。

中性冷灰外壳；各 stripe 颜色作点缀：

| 条纹色 | 界面用途 |
|--------|----------|
| 红 | 激活标记、强调 |
| 橙 / 黄 | 图表、「今日」、高亮 |
| 绿 | 字符串、行号标题 |
| 蓝 | 类型、间隔色 |
| 紫 | 关键字、光标、操作色 |
| 青 | 引用光晕、数字（深色） |

Dark LGBTQ 为深色底、同一色相族。有庆祝感，但不会把整屏涂成条纹。

### Lesbian（女同）

**含义：** 女性之间爱恋本身的温度与距离感。

像 **共用阳台上的暮色**：**陶土色**墙面余温、阴影里的 **灰玫瑰**、作为心跳而非符号的 **品红**。色板从暖桃外壳（约 25°）经 **落日橙** 间隔色、**腮红粉** 光晕，到关键字与光标的 **深玫瑰品红** — 亲密而不脆弱。Dark Lesbian 在炭灰底上保留余烬般暖光，而非霓虹。相较 LGBTQ 向外展开的色谱，Lesbian 停留在同一情绪频段：贴近、身体感、安静的自信。

### MS-DOS

**含义：** **微软 DOS** — 1980–90 年代经典 PC 文本模式。

饱和 **IBM 蓝** 背景（约 240°），**白** 前景，**黄** 字符串，**绿** 操作色，**青** 弱化文字 — 熟悉的 `DIR` / `AUTOEXEC.BAT` 观感。仅深色。复古终端情怀；不以护眼为首要目标。

### Matrix II（黑客帝国 II）

**含义：** 致敬电影 ***黑客帝国（The Matrix，1999）*** — 片名加 “II” 作续集式命名。

**霓虹绿**（约 135°）配 **近黑** 绿调面板。网格与边框用 subtle 绿色营造玻璃纵深；光标与激活项发光。字符串与函数偏亮绿；关键字为中绿。赛博朋克终端风 — 暗光环境下最佳。

### Country（各国文化色）

**含义：** 每一套色板取自 **传统文化中的经典色彩表达** — 山水、工艺、仪式与日常之美 — 而非国旗配色。乌克兰的天蓝与麦金是其中最直观的范例；其余国家沿同一思路展开。

| 主题 | 文化色彩叙事 | 底色 |
|------|--------------|------|
| **China 中国** | **水墨山水：** 宣纸白、墨与淡墨灰、远山 **花青**、偶现 **朱砂** 印色 — 烟岚、留白、层次。 | 浅色 |
| **Russia 俄罗斯** | **圣像与冬境：** 洋葱顶 **鎏金**、正教深红、**白桦** 皮白、雪上 **铅蓝** 暮光 — 肃穆、冷冽、内光。 | 浅色 |
| **USA 美国** | **旷野：** **草原天** 蓝、**麦浪** 琥珀、**谷仓** 红、磨旧的丹宁 — 边疆光线，不是庆典条幅。 | 浅色 |
| **Japan 日本** | **菊与刀：** 皇室 **绯**、**和纸** 白、**墨** 黑、克制灰 — 武士道式的清冽：美被纪律拉紧。 | 浅色 |
| **Ukraine 乌克兰** | **天空与麦浪（небо й пшениця）：** 开阔 **天蓝** 与成熟 **金黄** — 田野接夏空的地平线；质朴、辽阔、有希望。 | 浅色 |

| **Norway 挪威** | **峡湾夜：** 深 **海蓝**、**极光** 青绿点缀、**松影**、水面 **极昼** 琥珀 — 北欧深色、疏朗、亮边。 | 深色 |
| **UK 英国** | **雨与书房：** **雾灰**、**板岩**、**皮装书** 褐、 muted **玫瑰** — 湿石、下午茶时分、传世的工艺感。 | 浅色 |

主色渲染外壳；辅色与点缀用于字符串、光标、选区与 Git 标记。浅色款为淡洗背景；挪威、德国等深色款以文化中的「影」为底、亮色为边。

## 安装

**从扩展市场** — 在 VS Code 或 Cursor 中搜索 [Minor Themes](https://marketplace.visualstudio.com/) 并安装。

**从源码**

```bash
git clone https://github.com/lenik/minor-themes.git
cd minor-themes
pnpm install
pnpm run package
```

通过 **扩展 → … → 从 VSIX 安装…** 安装生成的 `.vsix` 文件。

## 使用

1. 打开命令面板（`Ctrl+Shift+P` / `Cmd+Shift+P`）。
2. 运行 **Preferences: Color Theme**（首选项：颜色主题）。
3. 在 **Minor Themes** 列表中选择主题。

## 开发

```bash
pnpm install
pnpm run generate   # 重新生成 themes/*.json
pnpm run package    # 打包 .vsix（发布前会自动执行 generate）
```
