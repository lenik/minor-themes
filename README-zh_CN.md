# Minor Themes

一组 VS Code 配色主题：柔和粉彩、骄傲旗帜色、复古终端风格，以及植根于各国传统文化经典的色彩表达。

## 主题说明

各主题的色板定义在 `scripts/generate-themes.mjs` 中，与 [UiTheme](https://github.com/lenik) 项目共用。浅色/深色成对主题为 **各自独立手写** 的 palette，并非由彼此推导。

### Innocent（纯真）

**含义：** *Innocent* — 纯真、清淡、不张扬。

仅提供浅色款。以腮红玫瑰与淡薰衣草为主（色相约 280–340°）：背景近乎白并带一层淡粉，正文为低饱和紫灰。强调色为灰玫瑰与淡紫 — 选区、光标、括号匹配都偏轻盈，不刺眼。语法高亮：关键字玫瑰色，字符串紫灰，数字略深的粉。适合想要安静、偏女性化、长时间阅读不累眼的编辑环境。

### Maiden（少女）

**含义：** *Maiden* — 少女；也指温柔、矜持、略带古典的浪漫气质。

**Light Maiden（浅色）** — 暖调淡粉底色（低饱和红、约 94% 明度），**深玫瑰红**正文，**金黄**高亮（今日标记、字符串、未跟踪文件等）。行号带一点橄榄绿，有老式信纸感 — 淡粉信纸配浓色墨迹。

**Dark Maiden（深色）** — **玫瑰金夜色**：暖酒梅紫外壳（非通用灰底），腮红白字，金色强调与鼠尾草行号。与浅色同一浪漫气质，像灯下静读。

### Girl（女孩）

**含义：** *Girl* — 年轻、鲜明、活泼（偏大胆自信，而非幼稚粉嫩）。

**Light Girl（浅色）** — 高饱和 **珊瑚红** 侧栏/面板（约 4–6°），白底，**热粉**引用块，**天蓝**间隔色。饱和红光标、青色选区、黄色「今日」标记。

**Dark Girl（深色）** — **霓虹卡哇伊之夜**：深洋红外壳、珊瑚光标、青绿间隔、粉紫引用光晕。保留珊瑚/粉/青的鲜明个性，为暗光环境调校，而非压低饱和度。

### Morandi（莫兰迪）

**含义：** 得名于意大利画家 **乔治·莫兰迪（Giorgio Morandi）** — 其静物画以灰调、低饱和、和谐统一著称。

**Light Morandi（浅色）** — 暖 **灰米色** 窗口（约 35°、低饱和），**蓝灰** 正文（约 230°）。强调色为灰青与灰粉，边框与网格柔和，对比度刻意克制。

**Dark Morandi（深色）** — **暮色灰调**：蓝灰橄榄外壳、炭灰上的灰青与鼠尾草高亮 — 仍是画廊式安静，适合长时间编码。

### LGBTQ

**含义：** 取自 **彩虹骄傲旗**（Gilbert Baker 设计）的 stripe 色系。

条纹色作点缀（非整屏涂条纹）：

| 条纹色 | 界面用途 |
|--------|----------|
| 红 | 激活标记、强调 |
| 橙 / 黄 | 图表、「今日」、高亮 |
| 绿 | 字符串、行号标题 |
| 蓝 | 类型、间隔色 |
| 紫 | 关键字、光标、操作色 |
| 青 | 引用光晕、数字 |

**Light LGBTQ（浅色）** — 中性冷灰外壳，条纹色相作可读强调。有庆祝感，但不铺满条纹。

**Dark LGBTQ（深色）** — **午夜骄傲**：深海蓝外壳，同一 stripe 色系在暗底上更亮 — 紫操作色、绿字符串、黄高亮、青数字。

### Lesbian（女同）

**含义：** 女性之间爱恋本身的温度与距离感。

**Light Lesbian（浅色）** — **共用阳台上的暮色**：暖桃外壳（约 25°）、**落日橙** 间隔、**腮红粉** 光晕、**深玫瑰品红** 关键字与光标 — 亲密而不脆弱。

**Dark Lesbian（深色）** — **暖烬之夜**：深酒梅紫外壳，橙粉余光在阴影里 — 余烬而非霓虹。与浅色同一情绪频段：贴近、身体感、安静的自信。（相较 LGBTQ 向外展开的色谱，Lesbian 停留在同一暖色带。）

### MS-DOS

**含义：** **微软 DOS** — 1980–90 年代经典 PC 文本模式。

饱和 **IBM 蓝** 背景（约 240°），**白** 前景，**黄** 字符串，**绿** 操作色，**青** 弱化文字 — 熟悉的 `DIR` / `AUTOEXEC.BAT` 观感。仅深色。复古终端情怀；不以护眼为首要目标。

### Matrix II（黑客帝国 II）

**含义：** 致敬电影 ***黑客帝国（The Matrix，1999）*** — 片名加 “II” 作续集式命名。

**霓虹绿**（约 135°）配 **近黑** 绿调面板。网格与边框用 subtle 绿色营造玻璃纵深；光标与激活项发光。字符串与函数偏亮绿；关键字为中绿。赛博朋克终端风 — 暗光环境下最佳。

## 安装

**从扩展市场** — 在 VS Code 或 Cursor 中搜索 [Minor Themes](https://marketplace.visualstudio.com/) 并安装。

**从源码**

```bash
git clone https://github.com/lenik/minor-themes.git
cd minor-themes
pnpm install
pnpm run package
```

通过 **扩展 → … → 从 VSIX 安装…** 安装生成的 `.vsix` 文件。

## 使用

1. 打开命令面板（`Ctrl+Shift+P` / `Cmd+Shift+P`）。
2. 运行 **Preferences: Color Theme**（首选项：颜色主题）。
3. 在 **Minor Themes** 列表中选择主题。

## 开发

```bash
pnpm install
pnpm run generate   # 重新生成 themes/*.json
pnpm run package    # 打包 .vsix（发布前会自动执行 generate）
```
